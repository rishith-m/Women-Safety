# Women's Safety App 🛡️

An AI-powered Android application designed for proactive women's safety.

## 🚀 Key Features
- **Voice SOS:** Automatically triggers alerts when keywords like "Help" or "SOS" are detected via AI.
- **Predictive Safe Routes:** Uses a Random Forest ML model to suggest the safest navigation paths.
- **Manual SOS:** 2-second hold button to prevent accidental alerts.
- **Offline Reliability:** Sends SMS with Google Maps location if the internet is unavailable.

## 🛠️ Tech Stack
- **Frontend:** Android (Java), Material 3, Google Maps API, Retrofit.
- **Backend:** Python (Flask), Scikit-Learn (Random Forest), MongoDB.
- **Deployment:** Render (Cloud).
